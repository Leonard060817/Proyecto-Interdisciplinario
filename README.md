# Sistema Integral de Gestión Clínica

  Proyecto que simula el desarrollo de un sistema para una clínica privada internacional.
  Permite administrar pacientes, turnos, historias clínicas, médicos, usuarios y reportes.





# Objetivos principales:

  - Optimizar la gestión interna de la clínica.
  - Asegurar la disponibilidad y confidencialidad de los datos.
  - Ofrecer una interfaz amigable y adaptable a distintos tipos de usuarios.
